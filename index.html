Sobre o curso

Este curso é voltado para iniciantes que desejam aprender lógica de programação de forma prática e objetiva.

O que você vai aprender

  • Conceitos básicos de lógica de programação
  • Algoritmos e fluxogramas
  • Estruturas de controle
  • Funções e procedimentos
  • Estruturas de dados

Pré-requisitos

Não é necessário ter conhecimentos prévios em programação. O curso é voltado para iniciantes.

Comece hoje mesmo! Aprender lógica é o primeiro passo para entrar no mundo da programação!
